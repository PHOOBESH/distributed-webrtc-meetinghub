# 🧠 SnakesLabs MeetingHub  
### Real-Time Distributed WebRTC Video Conferencing Platform

A scalable, real-time video conferencing platform built using **Python Flask**, **WebRTC**, and **WebSockets (Socket.IO)**, supporting multi-peer communication with ultra-low latency and AI-powered meeting intelligence.

---

## 🚀 Features

### 🎥 Real-Time Video Conferencing
- Peer-to-peer video/audio communication using WebRTC
- Supports 5+ concurrent participants per meeting
- Sub-200ms latency for smooth interaction

### 🔗 WebSocket-Based Signaling
- Implemented using **Socket.IO**
- Handles session negotiation (offer/answer/ICE candidates)
- Enables real-time communication between peers

### 🌐 Distributed Architecture
- Fully decentralized WebRTC mesh architecture
- Adaptive network handling for stable streaming
- Optimized for low bandwidth conditions

### 🖥️ Screen Sharing & Collaboration
- Real-time screen sharing support
- Seamless switching between camera and screen

### 🤖 AI-Powered Meeting Intelligence
- Speech-to-text transcription
- Automated meeting summarization
- Sentiment analysis
- Action item extraction
- Powered by **Google Gemini**

### ⚙️ Backend Engineering
- REST APIs built with Flask
- Modular and scalable architecture
- Linux-based deployment
- Version controlled using Git
- Automated testing integration
